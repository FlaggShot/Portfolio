FlaggBot

This project is a powerful visual automation tool. It leverages mss for efficient screen capturing and pyautogui for precise mouse and keyboard control. Image comparisons are highly optimized using cv2 with COLOR_RGB2BGR conversion, ensuring maximum efficiency. Managed through a neat, dark-themed Tkinter GUI, the entire application utilizes multiprocessing for robust and responsive performance across complex automation tasks.
